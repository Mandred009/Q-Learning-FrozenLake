Frozen Lake Q_LEARNING

Simple Demonstration of Q-Table

1) ENVIRONMENT
Open AI's FrozenLake-V0 for training and testing.

2) ACTION SPACE
0,1,2,3- Movement in only 4 directions.

3) OBSERVATIONS SPACE
The number of the grid.

4) POLICY
Q_Learning

5) REWARDS
+1 for end goal.
-1 for hole.
0 for others.
